# Hashup
**Yet another static sites generator**

## Installation

Add this line to your application's Gemfile:

    gem ins 'hashup'

And then execute:

    $ hashup setup myblog

And then:

    $ cd myblog
    $ hashup preview

There you go!

## Usage

Post a blog

    hashup post

Create a page
    
    hashup page
